# 微信自主摇一摇 — 项目日志

> 微信内加速度传感器 Hook Xposed 模块 + 摇一摇计数网页

## 项目概述

一个完整的摇一摇自动化工具链，包含：

- **前端网页** (`index.html`) — 手机端摇一摇计数游戏，通过 `DeviceMotionEvent` API 检测摇晃
- **Xposed 模块** (`LSPosed/`) — 基于 LSPosed 框架的 Android 模块，Hook 系统层加速度传感器数据

  - 方案一：Hook `SensorManager.registerListener` 4 个公开重载
  - 方案二：Hook `SystemSensorManager.registerListenerImpl`（5/6/7 参数变体）
  - 方案三：Hook Chromium WebView 传感器路径（`PlatformSensor.onSensorChanged`）
- 直接 Hook 监听器的 `onSensorChanged` 方法，拦截并修改真实数据
- 注入循环 + 真实数据拦截双保险机制
- 使用 `Unsafe.allocateInstance` 创建 SensorEvent（兼容所有 Android 版本）
- 新增悬浮窗实时控制（`FloatingControlService`）
- 新增设置界面 UI，速度 1~50 次/秒可调
- 配置写入 /data/local/tmp/，通过 su 权限全局共享

### v1.x

- 构建系统 AGP 8.2.0 + Gradle 8.5 + JDK 17 兼容性调试

## 构建说明

### 前置条件

- Android SDK（API 33+）
- JDK 17
- Gradle 8.5（项目自带 wrapper）
- Android 设备已 Root + 安装 LSPosed

### 构建命令

```bash
cd LSPosed
./gradlew assembleDebug
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

或运行 `build_and_install.bat`（Windows 一键打包安装）。

## 使用流程

1. 构建模块 → 生成 APK
2. 安装到 Root 手机
3. 打开 App → 设置速度 → 保存
4. LSPosed Manager 启用模块，作用域勾选微信
5. 重启微信
6. 打开摇一摇网页 → 开始挑战 → 自动计数

## 注意事项

- 模块仅在微信进程 (com.tencent.mm) 中生效
- 需要 Root 权限（Magisk + LSPosed）
- 速度范围 1~50 次/秒，建议 10~30 次/秒效果最佳
- 配置文件路径：`/data/local/tmp/shakecheat_config.txt`
